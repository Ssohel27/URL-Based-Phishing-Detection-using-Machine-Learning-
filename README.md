# URL-Based-Phishing-Detection-using-Machine-Learning-
Malicious URLs have become the primary attack vector for distributing malware and stealing credentials. With explosive growth in digital banking, e-commerce, and cloud services, users face increasingly sophisticated phishing websites that closely mimic legitimate ones.

The Solution: An intelligent, automated system that analyzes URLs and classifies them as safe or unsafe before users interact with them. This project implements such a system using Python (backend), Flask (web framework), and HTML/CSS/JavaScript (frontend), integrating machine learning models for real-time threat detection.
